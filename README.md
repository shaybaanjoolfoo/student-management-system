# Student Management System

A desktop application developed using Java Swing, SQLite, and JDBC to manage students, courses, and grades.

## Features
- Student CRUD (Add, Update, Delete, View)
- Course management
- Grade assignment with validation
- Search and filtering
- Reports (top students, averages)
- Theme support (Light/Dark)

## Technologies
- Java
- Java Swing
- SQLite
- JDBC

## What I Learned
- Object-Oriented Programming (OOP)
- DAO design pattern
- Database design and constraints
- GUI development with Swing
